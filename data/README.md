# Data

| File | Description |
|---|---|
| `enriched.csv` | Canonical dataset — 86,114 domains with severity, category, GeoIP, favicon hash |
| `high_severity.csv` | CRITICAL+HIGH subset — 20,480 domains |
| `dead_domains.csv` | Parked / dead domains enumeration |
| `clusters.json` | Operator cluster map — 2,939 infrastructure clusters |
| `all.txt` | All-time new registrations (auto-updated daily) |
| `index.json` | Daily new domain feed index (used by live Pages chart) |

→ IOC blocklists are in [`/ioc/`](../ioc/)
→ Daily new registrations are in [`/data/new/`](new/)
