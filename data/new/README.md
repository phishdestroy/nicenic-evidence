# Daily New Registrations

New domains registered with NICENIC (IANA #3765), organized by date.

```
new/
└── 2026/
    ├── 01/
    │   ├── 2026-01-15.txt   ← domains registered on Jan 15
    │   └── ...
    ├── 2026-01.txt          ← all January registrations
    └── ...
```

Each `.txt` file contains one domain per line, sorted alphabetically.
Updated daily by [GitHub Actions](../../.github/workflows/daily_fetch.yml) at 06:00 UTC.
