# Legal Notice

## Data Sources and Collection Methodology

All data in this repository originates from publicly accessible sources under
established legal frameworks.

**Zone File Access**  
Obtained through ICANN's Centralized Zone Data Service (CZDS). CZDS access
requires an approved application and is governed by ICANN's zone data access
agreement, which explicitly permits security research and abuse reporting.

**WHOIS Data**  
Queried via the standard WHOIS protocol (RFC 3912) against publicly accessible
WHOIS servers. WHOIS data is designated public by ICANN policy and each
registrar's obligations under the Registrar Accreditation Agreement (RAA).

**HTTP and DNS Data**  
HTTP fingerprinting, DNS resolution, and screenshot capture were performed
against URLs publicly reachable by any internet user without authentication.
No protected systems were accessed. No security controls were bypassed.

## No Non-Public Data

This repository contains no:

- Private user data or PII collected without consent
- Credentials, passwords, or authentication material
- Data obtained by unauthorized access to computer systems
- Information protected by privilege or confidentiality obligations

and IP addresses are network identifiers, not personal data under GDPR
Recital 30 in the context of abuse investigation (legitimate interest basis).
